Gene-Set Survival

Kaplan-Meier Survival Curves for a Gene-Set

  • Survival Analysis
    1. This program produces KM curves for a gene-set and metaplot for each subtype.
  • Parameters
    1. Id Type
      • Type of identifiers being submitted. Either HGU133A probesets (2046224_at) or a HUGO gene symbol as input from a file below. If gene symbol is selected the best available probeset will be identified using jetset Add link
    2. File
      • Text file with the first column containing the gene ids (Affy or Gene Symbol). See this Example
    3. Months
      • The number of months to plot. This parameter does not currenlty alter significance testing.
    4. Label
      • Name to be used as the id for analysis.
    5. Subtype
      • Survival plots will be generated for each tumor subtype from the chosen algorithm. PAM50 or MOD.
    6. Split
      • Method of splitting low and high groups. Median, Tertile or Quartile.
    7. Tamoxifen
      • Select patient cohort according to tamoxifen treatment. Yes, No or Compare across treatment.
    8. Chemo
      • Select patient cohort according to chemo treatment. Yes or No.
    9. End Point
      • Either Relapse-Free Survival (RFS), Distant-Metastasis Free Survival (DMFS), or the aggregate (RFS/DMFS).
  • Data Output
    1. Following submission, the results are printed in the webpage with a link to download all the relevent data and plots as a zip file. All plots are generated as PDFs to ensure high quality for publication purposes.Generated data include:
      1. KM curves for each subtype
      2. Meta-plots for each subtype
      3. All gene expression and associated clinical data
